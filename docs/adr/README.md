# Architecture Decision Records (ADR)

This directory stores short, immutable records of significant engineering decisions.

## Format

- File name: `NNNN-title.md`
- Status values: `Accepted`, `Superseded`, `Proposed`
- Sections:
  - Context
  - Decision
  - Consequences
